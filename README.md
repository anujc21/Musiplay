# Musiplay

## A music streaming application made using React and NodeJS using YouTube Search and YouTube IFrame APIs.

### Live @ https://musiplay.pages.dev/

### Run Locally

#### Server

#### 1. Run "npm install".
#### 2. Run "node start server.js".

#### Client

#### 1. Run "npm install".
#### 2. Run "npm run dev".
#### 3. Start the required port on browser.
